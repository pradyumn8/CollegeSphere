# CollegeSphere

CollegeSphere is a web application built to simplify and streamline various aspects of college and academic management. The app includes features to help students, faculty, and administrators manage tasks efficiently in a collaborative environment.

## Demo

Check out the live demo: [CollegeSphere on Netlify](https://collegesphare.netlify.app/)

## Features

- User-friendly interface for students and staff.
- Intuitive navigation with smooth scroll.
- Engaging animations using `framer-motion`.

## Installation
1. Clone the repository:

 ```bash
   git clone https://github.com/pradyumn8/CollegeSphere.git   
 ```
### Navigate into the project directory
 ```bash
 cd CollegeSphere
 ```

### Install dependencies
   ```bash
  npm install
   ```
### Run the Project
   ```bash
  npm run dev
   ```
#### Visit http://localhost:5153 in your browser.
